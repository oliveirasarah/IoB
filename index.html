O que é IoB?

A sigla vem do termo em inglês Internet of Behavior, ou Internet do Comportamento em português.

É uma forma de utilizar dispositivos digitais para trazer dados para o offline, e de trazer a tona o comportamento das pessoas, através de dados, para que seja possível trabalhar da melhor forma possível com esses dados.

De quais campos ela faz uso?

Tecnologia (para coleta de dados, dessa “poeira digital” deixada pelas pessoas).

Data analytics (a análise dos dados coletados).

Behavioral science (a ciência do comportamento humano).

Internet of Behavior (IoB) X Internet of Things (IoT)

A IoB vem sendo bastante utilizada em conjunto com a IoT.

O que é IoT?

Em suma, é a integração de diversos dispositivos que conectam-se com a internet e entre si.

Enquanto a IoT realiza a coleta de uma grande quantidade de dados, a IoB analisa como as pessoas se comportam através do rastros deixado por eles.

Os impactos do IoB para a comunicação e o marketing

Para o marketing, os dados ajudam a:


  • Entender as práticas dos usuários
  • Reforçar valores de marca
  • Melhorar a experiência do usuário
  • Gerar insights através da análise do comportamento dos usuários
